Expand description

Types that can be marked autocompletable in a slash command parameter.

Includes almost all types that can be used as a slash command parameter in general, except some built-in model types (User, Member, Role…)

Type of the partial input. This should be Self except in cases where a partial input cannot be parsed into Self (e.g. an IP address)
